# operacao-abismo
Um jogo de sobrevivência, tensão e exploração submarina desenvolvido em JavaScript utilizando o framework **Phaser 3**.   Nesta experiência, você pilota um submarino gravemente avariado nas profundezas do oceano. Com a navegação permanentemente danificada e imerso na escuridão total, sua única ferramenta é um sonar que consome bateria a cada pulso.
